  • Chord C
    Chord Cm

    For this chord, it requires you to use up frets 1 to 3, your first finger should be placed on string B, second finger on string D and the final finger on string A. This chord will require you to make sure that your fingers are not resting on any of the other strings because it will mess the sound up.

    Chord G

    This chord is a bit more challenging because of where you need to place your fingers. The first finger needs to be placed on the A string on the second fret, the second on the E string on the third fret with the third finger on the third fret on string E.
